# Hospital Management System (C++)

## Overview
A C++ OOP-based Hospital Management System that manages patients, doctors, and appointments with persistent file storage.

## Features
- Add / View / Search Patients
- Add / View / Search Doctors
- Schedule Appointments
- View Appointments
- File handling for data persistence

## Tech Stack
- C++
- Object-Oriented Programming
- STL (Vectors)
- File Handling

## Project Structure
- src/ → Source files
- include/ → Header files
- data/ → Stored records

## How to Run
```bash
g++ src/*.cpp -I include -o hospital
./hospital
